AES Encryption

ENCRYPT it!! supports encryption with the AES-128 algorithm. This algorithm uses a cipher key with length of 128 bits. To create the key, ENCRYPT it!! uses a derivation function based on an SHA-128 hash algorithm. A key derivation function produces a derived key from a text password defined by the user. To increase the cost of an exhaustive search for passwords, ENCRYPT it!! uses a big number of iterations to produce the cipher key from the text password.
